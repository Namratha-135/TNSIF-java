# 🛍️ Shopping Mall Management System

A simple Spring Boot REST API project for managing shop owner information in a shopping mall. The application performs CRUD (Create, Read, Update, Delete) operations using Spring Boot, Spring Data JPA, and PostgreSQL.

## 🚀 Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Postman

## ✨ Features
- Add a new shop owner
- View all shop owners
- Update shop owner details
- Delete shop owner records
- RESTful API implementation
- PostgreSQL database integration

## 📌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/saveshopowner` | Add a new shop owner |
| GET | `/getshopowner` | Retrieve all shop owners |
| PUT | `/updateshopowner/{id}` | Update shop owner details |
| DELETE | `/deleteshopowner/{id}` | Delete a shop owner |

## ▶️ How to Run
1. Clone the repository.
2. Configure PostgreSQL in `application.properties`.
3. Run the Spring Boot application.
4. Test the APIs using Postman.
